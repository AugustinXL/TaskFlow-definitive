# TaskFlow

Projeto Node.js simples para gerenciar tarefas (TaskFlow).

## Estrutura

- src/: código fonte
- tests/: testes unitários e de integração
- jmeter/: arquivos de exemplo para testes de estresse
- package.json: scripts úteis

## Como rodar

1. `npm install`
2. `npm run dev` - roda em desenvolvimento (nodemon)
3. `npm start` - roda em produção

## Testes

- `npm test` — todos os testes
- `npm run test:unit` — testes unitários
- `npm run test:integration` — testes de integração

## JMeter (simulação)

- `npm run jmeter:print` — imprime resultados simulados do teste de estresse
